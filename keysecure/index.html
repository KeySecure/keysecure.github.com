KeySecure

Secure password storage on your phone.

Features

  • View, Edit, and Create your secure informations, Almost any date type supported

  • Sync files to iCloud and Dropbox using the Dropbox iOS app

  • Copy password entries to the clipboard for easy entry

  • Open websites in Safari while KeySecure runs in the background

  • iOS 8 features: (1). App Extension supported; (2). Apple Touch ID supported; (3). Apple Watch supported

  • iOS 9 features: (1). Spotlight searching supported; (2). Multitasking split views supported for iPad Pro
